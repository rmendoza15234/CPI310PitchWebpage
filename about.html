About Project Page

This website allows users to play a simple adventure platformer right from their browser. Players can log in to save their game clear time and compete with other users to achieve the fastest time. Users are welcomed with a start page, where they can read what the game is about and also see the top 10 fastest clear times from other players. The game will feature multiple stages in which the player must complete as fast as possible to reach the end of the game. Besides the game, the website will feature multiple pages in which the user can traverse to. These include an “about us” section, login page, how to play page and a highscores page.
